# Active Directory Lab

## 🎯 Objective

The goal of this lab is to learn how to deploy and manage a basic Active Directory environment using Windows Server and a Windows client machine.

This project is part of my learning journey toward becoming an IT Support Technician.

---

## 🖥️ Lab Environment

- VirtualBox
- Windows Server
- Windows client machine
- Internal / host-only network
- Active Directory Domain Services
- DNS

---

## 📚 Planned Steps

- [ ] Create the virtual machines
- [ ] Install Windows Server
- [ ] Configure a static IP address
- [ ] Install the Active Directory Domain Services role
- [ ] Promote the server as a Domain Controller
- [ ] Create a new domain
- [ ] Create Organizational Units
- [ ] Create users
- [ ] Join a Windows client to the domain
- [ ] Test user authentication
- [ ] Document issues and solutions
