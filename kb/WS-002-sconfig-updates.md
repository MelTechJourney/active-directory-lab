# WS-002 - Installation des mises à jour avec SConfig

## Symptôme

Impossible de savoir quelle option utiliser pour installer les mises à jour.

## Cause

Première utilisation de SConfig.

## Résolution

Utiliser :

```text
T
```

pour installer toutes les mises à jour.

## Vérification

Le téléchargement puis l'installation des mises à jour démarrent correctement.

## Mots-clés
